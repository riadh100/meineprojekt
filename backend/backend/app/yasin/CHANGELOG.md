# Versionsschema

```
MAJOR.MINOR.PATCH
```

Beispiel:

```
9.0.0
```

- **MAJOR** – inkompatible Änderungen
- **MINOR** – neue Funktionen
- **PATCH** – Fehlerbehebungen

---

# Änderungsarten

- **Added** – Neue Funktionen
- **Changed** – Änderungen
- **Deprecated** – Veraltet
- **Removed** – Entfernt
- **Fixed** – Fehlerbehebungen
- **Security** – Sicherheitsupdates

# Ziel

Das Changelog dokumentiert transparent die Entwicklung von **Yasin AI** und ermöglicht Entwicklern und Anwendern, Änderungen zwischen Versionen schnell nachzuvollziehen.
