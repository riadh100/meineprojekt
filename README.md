# AI Empire Pro V8

Eine modulare KI-Plattform für Trading, Automatisierung, Telegram, Video, Game Hub und AI-Assistent.

---

# Features

- Dashboard
- AI Assistant
- Trading Center
- Telegram Bot Manager
- Video Studio
- Game Hub
- Tools Center
- Setup
- REST API
- Node.js Backend
- MongoDB Support
- JWT Authentication
- OpenAI Integration
- Local Storage
- Backup & Restore

# Installation

## Frontend

Einfach den Projektordner mit einem Webserver starten.

Beispiel:

```
Live Server
```

oder

```
python -m http.server
```

---

## Backend

```
cd backend
```

Pakete installieren

```
npm install
```

Server starten

```
npm start
```

Entwicklungsmodus

```
npm run dev
```
